Baixar PDF do Google Drive (Somente Leitura)

Baixe documentos que só podem ser visualizados — o script junta tudo em um único PDF

Use apenas em documentos que você tem permissão legítima para visualizar. O conteúdo já está visível na sua tela — este script apenas salva localmente.
Método 1: Console do Navegador
Método 2: Botão na Barra de Favoritos (recomendado)
1

Abra o PDF no Google Drive

Acesse o link do PDF e espere ele carregar por completo na tela
2

Abra o Console do navegador

Pressione a tecla F12 no teclado. Vai abrir um painel na tela. Clique na aba Console.
3

Copie o script abaixo e cole no Console

Clique no botão "Copiar", depois clique dentro do Console e pressione Ctrl+V 
(async function(){
  /* Carrega a biblioteca jsPDF para gerar o PDF final */
  if(!window.jspdf){
    const s=document.createElement('script');
    s.src='https://cdnjs.cloudflare.com/ajax/libs/jspdf/2.5.1/jspdf.umd.min.js';
    document.head.appendChild(s);
    await new Promise(r=>s.onload=r);
  }
  const{jsPDF}=window.jspdf;

  const urls=new Set();
  const sleep=ms=>new Promise(r=>setTimeout(r,ms));

  function collect(){
    document.querySelectorAll('img').forEach(img=>{
      const s=img.src||'';
      if((s.includes('googleusercontent')||s.includes('lh3')||s.includes('ggpht'))&&img.naturalWidth>200){
        urls.add(s);
      }
    });
    document.querySelectorAll('[style*="background"]').forEach(el=>{
      const bg=getComputedStyle(el).backgroundImage;
      if(bg&&bg.includes('url(')){
        const m=bg.match(/url\(["']?(.*?)["']?\)/);
        if(m&&(m[1].includes('googleusercontent')||m[1].includes('lh3')))urls.add(m[1]);
      }
    });
  }

  const container=document.querySelector('.ndfHFb-c4YZDc-q77wJc')
    ||document.querySelector('[role="document"]')
    ||document.body;
  let lastH=0;

  console.log('Rolando o documento... nao feche esta aba.');
  for(let i=0;i<300;i++){
    collect();
    container.scrollBy(0,window.innerHeight*0.8);
    await sleep(700);
    if(container.scrollHeight===lastH&&i>3)break;
    lastH=container.scrollHeight;
  }

  container.scrollTo(0,0);
  await sleep(500);
  collect();

  const arr=[...urls].sort();
  console.log('Encontradas '+arr.length+' paginas. Baixando e montando o PDF...');

  const pdf=new jsPDF({unit:'px',hotfixes:{px_scaling:1}});
  let first=true;

  for(let i=0;ir.blob());
      const dataUrl=await new Promise((res,rej)=>{
        const fr=new FileReader();
        fr.onload=()=>res(fr.result);
        fr.onerror=rej;
        fr.readAsDataURL(blob);
      });
      const img=new Image();
      img.src=dataUrl;
      await new Promise(r=>img.onload=r);

      const w=img.naturalWidth,h=img.naturalHeight;
      if(first){
        pdf.addPage([w,h],'png');
        pdf.addImage(dataUrl,'PNG',0,0,w,h);
        first=false;
      }else{
        pdf.addPage([w,h],'png');
        pdf.addImage(dataUrl,'PNG',0,0,w,h);
      }
      console.log('Pagina '+(i+1)+' de '+arr.length+' processada.');
    }catch(e){
      console.warn('Falha na pagina '+(i+1)+': '+e.message);
    }
  }

  /* Remove a primeira pagina em branco que o jsPDF cria automaticamente */
  if(pdf.getNumberOfPages()>1){
    pdf.deletePage(1);
  }

  pdf.save('documento_baixado.pdf');
  console.log('Pronto! PDF salvo com '+arr.length+' paginas.');
})();
Script copiado! Agora cole no Console e pressione Enter.
4

Pressione Enter e espere

O script vai rolar o documento sozinho, baixar as páginas e montar um único arquivo documento_baixado.pdf. Não feche a aba até terminar.
O que é este método? Você vai criar um botão na barra de favoritos do navegador. Depois, toda vez que abrir uma prova no Google Drive, basta clicar nesse botão para baixar o PDF completo. Você só precisa fazer a instalação uma única vez.
1

Mostre a barra de favoritos do navegador

A barra de favoritos é aquela faixa branca logo abaixo do endereço do site, onde ficam os sites que você salva. Se ela não estiver visível:
No Chrome: clique nos três pontos (⋮) no canto superior direito > Favoritos > marque Mostrar barra de favoritos.
Atalho rápido: pressione Ctrl + Shift + B no teclado.
2

Crie um novo favorito qualquer na barra

Clique com o botão direito do mouse em qualquer espaço vazio da barra de favoritos e escolha "Adicionar página..." (ou "Adicionar favorito"). Vai abrir uma janelinha.
3

Dê um nome ao favorito

No campo Nome, digite: Baixar Prova (ou outro nome que você queira). O nome não importa, é só para você identificar o botão.
4

Cole o código no campo URL e salve

No campo URL (ou endereço), apague tudo que estiver lá. Depois clique no botão "Copiar" aqui embaixo, volte para a janelinha do favorito, clique no campo URL e pressione Ctrl + V para colar. Por fim, clique em Salvar (ou Concluir).

5

Abra a prova no Google Drive

Acesse o link da prova que você quer baixar e espere ela carregar por completo na tela.
6

Clique no favorito "Baixar Prova" que você criou

O botão está na barra de favoritos, no topo do navegador. Ao clicar, o script começa a rodar automaticamente: ele vai rolar a prova sozinho, página por página, e quando terminar vai salvar o arquivo documento_baixado.pdf na sua pasta de Downloads.
7

Espere terminar e pronto!

Não feche a aba do navegador enquanto o script estiver rodando. Quando ele terminar, o PDF completo será baixado automaticamente. Para baixar outra prova no futuro, basta repetir os passos 5 e 6 — o favorito já está pronto.
Importante saber:
  • O script rola o documento automaticamente — pode levar alguns minutos em documentos longos
  • Não feche a aba do navegador durante o processo
  • O resultado final é um único arquivo documento_baixado.pdf com todas as páginas
  • As páginas são salvas como imagens dentro do PDF (não como texto selecionável)
